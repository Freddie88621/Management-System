💾 数据管理

💾 保存为独立文件（推荐）

将当前所有数据嵌入到HTML文件中，复制到任何电脑打开后数据完整保留，无需网络或数据库。

📤 导出备份

将数据导出为JSON备份文件或Excel文件，可在不同设备间迁移。

📥 导入数据

从JSON备份文件或Excel文件（由本系统导出的格式）恢复/追加数据。Excel导入为追加模式，JSON导入将覆盖全部数据。

🗑 清空数据

清除所有任务和阶段数据，重置系统。操作不可撤销，请先导出备份！

☁ 云端同步（跨电脑使用）

使用免费的 JSONBin.io 实现多电脑数据同步。步骤：① 注册 jsonbin.io 免费账号 ② 创建一个 Bin（内容填 {}）获得 Bin ID ③ 在 Account → API Keys 获取 Master Key ④ 填入下方并保存设置。之后每次"保存数据"都会自动推送到云端，换电脑后点"从云端拉取"即可同步最新数据。
